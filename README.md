<h1>React Assignment</h1>
<p>In this assignment React, Vite and Redux are used to make an Online Library Management System which incorporates basic functionalities of adding a new book, viewing more detailed description of book, category vise sorting, book search,etc. </p>

<h1>Objective of this assignment are:</h1>
<ol>
    <li>To develop home page containg popular books and category list of books </li>
    <li>To use react for bulding the application</li>
    <li>To build Browse Book page that includes functionalities such as: Book search, dynamic routing by category </li>
    <li>To develop Add Book page where new book can be added to library</li> 
    <li>To use Redux to manage state of the books list</li>
</ol>

<h1>How to deploy and run this application</h1>
<ul>
    <li>How to clone the application: git clone https://github.com/rijulsrivastava/Online-Library-System-Assignment.git </li>
    <li>Then open terminal</li>
    <li>run command: npm install</li>
    <li>locate src folder in the application</li>
    <li>run command: npm run dev</li>
</ul>
